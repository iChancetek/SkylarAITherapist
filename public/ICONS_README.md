# iSkylar PWA Icons

The PWA requires two icon sizes:
- `public/icon-192.png` (192x192 pixels)
- `public/icon-512.png` (512x512 pixels)

## Quick Icon Creation

You can create these icons using:

### Option 1: Online Tool
1. Go to https://favicon.io/favicon-generator/
2. Text: "iS"
3. Background: Purple gradient (#8b5cf6)
4. Font: Bold, modern
5. Download and rename to icon-192.png and icon-512.png

### Option 2: Design Tool (Figma/Canva)
1. Create 512x512 canvas
2. Purple-to-blue gradient background
3. White brain icon or "iSkylar" text
4. Export as PNG
5. Resize to 192x192 for smaller version

### Temporary Placeholder
Until you have proper icons, the manifest will still work but won't show custom icons when installing.

Place the final icons in:
- `public/icon-192.png`
- `public/icon-512.png`
